Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de:
- Praticar HTML e CSS na construção de interfaces
- Simular o visual de um serviço de streaming
- Evoluir conhecimentos em front-end através de um projeto prático

- Tecnologias utilizadas
- *HTML5*
- *CSS3*

- Funcionalidades
- Interface inspirada em plataformas de streaming
- Catálogo de filmes/séries organizado em cards
- Estrutura visual clara e intuitiva
